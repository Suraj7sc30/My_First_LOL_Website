HTML

Basics:-

1.HTML Headings:-

HTML Headings are the title or subtitle that you want to display on web page.
It is defined with < h1 to h6 > tags.
< h1 > defines the most important headings.
< h6 >- defines the least important headings.


2.HTML Links:-

HTML Links are defined with the <'a'> tag.
Example:< a href ="https://www.w3shool.com">This is a link'
This is a website link
This is a youtube link
The links destination is specified in the "href" attribute.

3.HTML Images:-

It is defined with < img > tag.
The source file (src), alternative text (alt),width,height are provide as attrbute.
Where alt specify extra infornation of img when network connection is slow or error occur.
Example: < img src="wp4221520-royal-enfield-continental-gt-650-wallpapers" alt="Royal Enfield width="111" height="211">

4.Empty Html Elements:-

Html Elements with no content are called empty elements.
< br > tag define a line break and it is empty element without closing tag
